Things I've Built

Mock Trading App

Tech Stack:

Used Flask to create a mock trading account web app. The app first prompts the user to create an account. Once logged in, the user is gifted an imaginary $10,000 with which to "buy" and "sell" stocks. Live stock prices are used by querying the IEX API. From their account, the user can manage their portfolio, review their transaction history, or deposit more imaginary currency.

Calculator App

Tech Stack:

This was built in practice of vanilla JavaScript and DOM manipulation. The app easily handles arithmetic opertions and has all of the functionality of a generic calculator. I used event listener methods to call specific functions based on user input. An intuitive design was created by styling a familiar user interface using CSS Grid.

Portfolio

Tech Stack:
